What is this?

This link hosts a machine learning model created using Teachable Machine, a tool that makes it easier for anyone – teachers, students, artists, makers of all kinds – to train machine learning models.

How does it work?

Machine learning models are trained on examples (e.g., images, sounds, poses) gathered by the creator. Their results depend on the data they’ve been trained on.
